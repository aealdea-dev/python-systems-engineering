# Linked List Implementation (Memory Management Study)

## Purpose
Custom implementation of a Singly Linked List to analyze pointer manipulation, memory traversal, and node-based data storage versus contiguous arrays.

## Key Learning Outcomes
- **Pointer Logic:** Implemented node-to-node traversal and bypass-style deletion.
- **Complexity Analysis:** Comparison of $O(n)$ traversal in Linked Lists vs. $O(1)$ indexing in Arrays.
- **Edge Case Management:** Explicit handling of `head` node reassignment during deletion.
